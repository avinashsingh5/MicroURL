# URL Shortener - Frontend

React + Vite + Tailwind CSS frontend for URL shortening service.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file:
```env
VITE_API_URL=http://localhost:5000
```

3. Run development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```
